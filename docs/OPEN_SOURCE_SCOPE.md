# Open Source Scope

This repository is the public website and docs surface for PARA.

## In Scope

- SvelteKit apps that render the public website
- public docs and schema reference content
- shared UI tokens and schema packages used by the site
- build and deploy instructions for the public website

## Out of Scope

- private app distribution flows
- private backend credentials
- local-only demo tunnels
- unpublished product experiments that are not ready for public review
- internal runbooks that would reduce operational safety if published verbatim

## Practical Rule

If a file would help an external contributor build, review, or improve the public website, it probably belongs here.

If a file exists mainly to operate private infrastructure or temporary demos, it probably does not.
